# Documentação do Achilles

## Visão Geral

Achilles é um gerenciador de jogos para Arch Linux com Hyprland (Wayland). Permite instalar jogos via Wine/Proton a partir de repacks (FitGirl, etc.) e organizar uma biblioteca pessoal com controle de tempo jogado, configuração por jogo e detecção automática de runtimes.
